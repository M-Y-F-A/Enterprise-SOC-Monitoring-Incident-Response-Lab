# Detection Use Cases

## Brute Force Detection
- Repeated failed authentication attempts over short windows
- Username targeting from single source
- Followed by eventual successful authentication

## Lateral Movement Detection
- Remote administration channels and service execution patterns
- Access to administrative shares
- Cross-host process creation anomalies

## Authentication Anomaly Detection
- Unusual logon types for user role
- New host/source combinations
- Time-based deviations from normal behavior
